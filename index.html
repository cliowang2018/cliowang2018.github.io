获取参数步骤：
1. 打开 console.firebase.google.com，用 Google 账号登录
2. 点击「新建项目」→ 输入名称（如 swim-clio）→ 完成创建
3. 点击「</> Web」图标注册应用 → 复制 firebaseConfig 里的参数填入下方
4. 左侧菜单 → Firestore Database → 新建数据库 → 选「测试模式」→ 完成
